## File type


<p>_config.yml: Main configuration of the site.</p>
<p>_data/navigation.yml: Defines navigation links.</p>
<p>_includes/header.html: Contains the HTML code for the header and navigation.</p>
<p>_posts/: Contains the blog posts.</p>
<p>_layouts/: Contains the page layouts.</p>
<p>index.html: The homepage of the site.</p>
